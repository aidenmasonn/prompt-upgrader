# Prompt Upgrade Rubric (V1)

When upgrading a prompt, always try to add:
1) Clear objective (one sentence)
2) Constraints / context (include relevant saved Q&A constraints)
3) Output format (explicit structure, e.g. bullets/table/json)
4) Evaluation criteria (what "good" looks like)
5) Assumptions (state them)
6) If useful: examples (1 short example)

Return:
A) Upgraded prompt
B) 3-6 bullet rationale
C) "Used memory" list: IDs/titles of Q&A items incorporated
